Why this project?

Our group brainstormed for a while and then decided, a restaurant website will be suitable for the project as it is neither too static nor too dynamic. Since it will be our first web development project the idea seemed great as we will be able to use what we learned in the class in this project.
Due to COVID-19 restaurants have been greatly affected. Since the restaurants can no longer allow people to dine in, we decided to add an ordering system to our website.
Using our delivery system people can order food then our fictional restaurant can deliver the required order to the same individual. Thus, ensuring a contactless, safe and hygienic food delivery.

Website layout

We decided to build our website in tie with the modern design trends. You will notice that our webiste has sort of a minimalistic feel to it and is also very modern.
Every component is made from scratch using css and no css frameworks have been used.
